Support & Help

Getting Started

Welcome to our Discord Moderator bot! This guide will help you get started and troubleshoot any issues you might encounter.

Quick Setup Guide

  1. Add the Bot: Click the "Add to Discord" button and authorize the bot for your server
  2. Set Permissions: Make sure the bot has the necessary permissions (Manage Messages, Kick Members, etc.)
  3. Test Commands: Try using !help to see all available commands
  4. Configure Settings: Use !prefix to set your preferred command prefix

Common Issues & Solutions

Bot doesn't respond to commands

Solution: Check if the bot has the "Send Messages" permission and is online. Also verify you're using the correct prefix.

Commands don't work as expected

Solution: Ensure the bot has the required permissions for the specific command. Check the bot's role hierarchy.

Bot can't mute/kick/ban users

Solution: The bot's role must be higher in the server's role hierarchy than the users it's trying to moderate.

Bot is offline

Solution: This is rare, but if it happens, please report it in our Discord server. The bot should be online 24/7.

Required Permissions

For the bot to function properly, it needs these permissions:

  • Send Messages: To respond to commands
  • Manage Messages: To delete messages and use clear command
  • Kick Members: To kick users from the server
  • Ban Members: To ban users from the server
  • Manage Roles: To mute users by adding/removing roles
  • Read Message History: To access previous messages

Still Need Help?

If you're still experiencing issues after trying these solutions, please:

  • Join our Discord server for live support
  • Check the Commands page for detailed usage information
  • Contact us through the Kontakt page
