Bot Commands

Available Commands

Here are all the commands you can use with our Discord Moderator bot. All commands are designed to be simple and effective.

!help

Get information about all available bot commands and their usage.

Usage: !help [command]

!mute

Mute a user for a specified duration. The user won't be able to send messages or join voice channels.

Usage: !mute @user [duration] [reason]

!warn

Issue a warning to a user. Warnings are logged and can be reviewed by moderators.

Usage: !warn @user [reason]

!prefix

Change the bot's command prefix for your server. Useful for avoiding conflicts with other bots.

Usage: !prefix [new_prefix]

!kick

Kick a user from the server. They can rejoin if they have an invite link.

Usage: !kick @user [reason]

!ban

Ban a user from the server. They cannot rejoin unless unbanned.

Usage: !ban @user [reason]

!clear

Delete a specified number of messages from the current channel.

Usage: !clear [number]

!info

Display information about a user, including their join date and roles.

Usage: !info @user

Permission Requirements

Most commands require appropriate permissions to use. Make sure the bot has the necessary permissions in your server:

  • Manage Messages
  • Kick Members
  • Ban Members
  • Manage Roles
  • Send Messages
