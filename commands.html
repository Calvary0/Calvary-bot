Bot Commands

Available Commands

Here are all the commands you can use with our Discord Moderator bot. All commands are designed to be simple and effective.

!help

Get information about all available bot commands and their usage. Use this command to see the full list of available commands or get detailed help for a specific command.

Usage: !help [command]

!ban

Permanently ban a user from the server. Banned users cannot rejoin unless manually unbanned by a server administrator. This is the most severe form of moderation.

Usage: !ban @user [reason]

!kick

Remove a user from the server temporarily. Kicked users can rejoin if they have a valid invite link. This is useful for temporary removal without permanent consequences.

Usage: !kick @user [reason]

!mute

Temporarily mute a user for a specified time period. Muted users cannot send messages in text channels or speak in voice channels until the mute expires.

Usage: !mute @user [time]

!unmute

Remove the mute status from a user immediately. This command can be used to unmute users before their mute duration expires.

Usage: !unmute @user

!warn

Issue a formal warning to a user. Warnings are logged in the bot's database and can be reviewed by moderators. Multiple warnings can lead to automatic actions.

Usage: !warn @user [reason]

!warnings

View all active warnings for a specific user. This command shows the total number of warnings and their details, helping moderators make informed decisions.

Usage: !warnings @user

!clear

Delete a specified number of messages from the current channel. This command helps keep channels clean by removing spam or inappropriate content.

Usage: !clear

!clear @user

Delete a specified number of messages from a specific user in the current channel. This is useful for removing spam from a particular user while keeping other messages.

Usage: !clear @user

!prefix

Change the bot's command prefix for your server. This is useful for avoiding conflicts with other bots that might use the same prefix.

Usage: !prefix [new_prefix]

!info

Display detailed information about a user, including their account creation date, server join date, roles, and other relevant details.

Usage: !info @user

Permission Requirements

Most commands require appropriate permissions to use. Make sure the bot has the necessary permissions in your server:

  • Manage Messages
  • Kick Members
  • Ban Members
  • Manage Roles
  • Send Messages
